# About

A Brainfuck compiler that compiles the program to x86 machine code, and runs it.

Optimizations:
* `+-><` folding
* find 0 optimization (ex `[<<<<]`, `[>>>>>]`)
* set 0 optimization (ex `[-]`, `[-]>[-]`)
* quick operation optimization (ex `>>>+<<<`, `<<<<--->>>>`)
* move-add/sub optimization (ex `[-<<<+>>>]`, `[->>>>-<<<<]`)

You can download the latest version in [releases](https://github.com/skejeton/BrainfuckJIT/releases).

# Building

Only works on Windows, should be easy to port to other systems by replacing `VirtualAlloc` with `mmap`, PR would be welcome.

Use `build.bat` to compile, or just `cl main.c`. You need to compile from x86 Native Tools Command Prompt for VS. Make sure it's 32-bit, not 64-bit.
